Learn how to quickly get started with Configu.
We'll use the Configu's CLI and the Configu's Saas Configuration Store to decalre, initalize, and export a simple set configuration data.

### 1. Install the Configu CLI

<CliInstallSnippet />

### 2. Declare a Config schema

```shell
configu init --getting-started
```

### 3. Use a Config Store

> Create a new organization in the [Configu SaaS Configuration Store](https://app.configu.com/).

```shell
configu store upsert --type "configu"
```

### 4. Upsert Config values to a "test" Config Set

```shell
configu upsert \
--store "configu" --set "test" --schema "./getting-started.cfgu.json" \
--config "GREETING=hey" --config "SUBJECT=<YOUR_NAME>"
```

### 5. Export the configuration data

```shell
configu export \
--store "configu" --set "test" --schema "./getting-started.cfgu.json" \
--format "Dotenv" \
> ".env"
```

and you will get a .env file for the "getting-started" ConfigSchema configurations from the "test" ConfigSet.

```.env
GREETING=hey
SUBJECT=<YOUR_NAME>
BAZ=hey, <YOUR_NAME>!
```

That's it! You should now have a simple set of configuration data stored in the Configu SaaS Configuration Store, ready to be used in your application.

To learn more about Configu and how to use it, you can continue to expolre the documentation and join the Configu [Discord channel](https://discord.com/invite/cjSBxnB9z8), where you can ask questions and get help from the Configu team and other community members.
