title: configu test
---

verify credentials and write access to a [ConfigStore](/config-store)

```shell
configu test [flags]
```

### Flags

- `--store <type>` (**required**) the ConfigStore to test.
- `--clean` (**optional**) cleans after test is done.

### Examples

```shell
configu test --store 'configu'

configu test --store 'configu' --clean
```
