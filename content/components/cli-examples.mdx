### CLI Upsert

<>
  <pre>
    <code>
      configu upsert --set "example" --schema "example.cfgu.json" --store "{props.scheme}://
      {props.uriExample}" --config "FOO=value" --config "BAR=bar"
    </code>
  </pre>
</>

### CLI Export

<>
  <pre>
    <code>
      configu export --set "example" --schema "example.cfgu.json" --store "{props.scheme}://
      {props.uriExample}" --format "Dotenv"
    </code>
  </pre>
</>

### CLI Delete

<>
  <pre>
    <code>
      configu delete --set "example" --schema "example.cfgu.json" --store --store "{props.scheme}://
      {props.uriExample}"
    </code>
  </pre>
</>
